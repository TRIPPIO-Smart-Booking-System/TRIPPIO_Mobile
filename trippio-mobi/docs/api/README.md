# API Documentation

Thư mục này chứa tài liệu API của Trippio.

## Files

- `swagger.json` - Swagger/OpenAPI specification file cho Trippio API

## Sử dụng

Bạn có thể:
1. Import file `swagger.json` vào Swagger UI để xem documentation
2. Sử dụng file này để generate API client code
3. Tham khảo khi tích hợp với backend API

## API Base URL

Production: `https://trippio.azurewebsites.net`

## Xem tài liệu online

Swagger UI: https://trippio.azurewebsites.net/swagger/index.html
